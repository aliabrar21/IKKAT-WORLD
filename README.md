# Pochampally Ikat - Premium Luxury Handloom Website

A premium luxury website for a Pochampally Ikat brand featuring authentic handloom sarees, custom orders, and a digital catalogue system.

## Features

- **Home Page**: Hero section, brand story, featured collection, why choose us, and WhatsApp CTA
- **Collection Page**: Filterable product grid with categories (Silk Ikat, Cotton Ikat, Dupattas, Lehengas)
- **Product Detail Page**: Image gallery, detailed specifications, availability status, and enquiry/pre-order options
- **Weaving Process Page**: Educational storytelling about Ikat weaving tradition
- **Custom Order Page**: Form for custom color/design requests
- **Contact Page**: Contact information, business hours, and Google Maps integration

## Tech Stack

- React.js (Functional Components)
- React Router DOM
- CSS (Custom styling - no Tailwind)
- Vite (Build tool)

## Design

- **Color Palette**: Soft beige (#F5F1E8), cream (#FAF8F3), maroon (#8B2635)
- **Typography**: Playfair Display (serif) for headings, Lato (sans-serif) for body
- **Style**: Premium luxury aesthetic with smooth animations and hover effects
- **Responsive**: Mobile-first design, fully responsive across all devices

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable components (Header, Footer, ProductCard, Button)
├── pages/           # Page components (Home, Collection, ProductDetail, etc.)
├── styles/          # Global CSS files
├── data/            # Mock product data (products.json)
└── App.jsx          # Main app component with routing
```
